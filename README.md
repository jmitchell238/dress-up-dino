# Dress-Up Dino

Dress a friendly dino with hats, scarves, glasses, and spots. Save favorites, show off with confetti. Creative free play for ages **4–6**.

**Play:** https://jmitchell238.github.io/dress-up-dino/

Part of [Arcade Hub](https://jmitchell238.github.io/arcade-hub/).

## Modes

| Mode | What |
|------|------|
| Free Play | Tap cosmetics forever — no goals |
| Match Me | Copy the little ghost outfit for a cheer |

## Features

- Categories: Color, Hats, Scarves, Glasses, Spots
- Save up to 8 favorite outfits (localStorage)
- Surprise random look + Show off dance/confetti
- Sound mute + reduced motion
- Offline PWA after first visit
- Zero fail screens

## Stack

Static HTML / CSS / Canvas. No build step.

## Versioning

`GAME_VERSION` in `js/config.js` ↔ `CACHE` in `sw.js`.

## Local preview

```bash
python3 -m http.server 8080
```

## Parents

No lives, ads, accounts, or fail screens. Creative play only.
